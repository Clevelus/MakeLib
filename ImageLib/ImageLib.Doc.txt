

Это библиотека для работы с образами

Эта библиотека ТОЛЬКО для внутренненнго пользования.


Есть классы 

CIMAGE -- Это класс который асоциируется с образом. Можно открыть существующий или создать новый. 
Внутри себя содержит секции, импорты, экспорты.

Позволяет:

Для вновь созданого образа :

- Добавлять секции
- Добавлять экспорты
- Добавлять импорты
- Перемещать ( менять виртуальные адреса ) секции.
- Строить бинарный слинкованный образ
- Сохранять бинарный образ

Для любых образон и новых и старых позволяет:
- Перечислять секции
- Перечислять экспорты
- Перечислять импорты

Позволяет Перемещать в виртуальном адрессном пространстве секции. 
 
CEXPORS

 Представляет собой поименнванный набор точек входа. 

CIMPORTS

 Представляет собой набор функций из внешних бибиотек.

CRELOCS
 
 Внутренний класс - не трогайте его. Он создается при построинии бинарного образа. И он есть у старого образа загруженного.

CSECTION
 
 Секция это кусок вируального адрессного пространства. Секии использутся системой при создании образа в памяти в виде набора страниц. 
 Секция всегда должна быть выровнена. 
 Секции в образе по виртуальным адресам должны идти непрерывно. Первая секция должна начинается с 0 но выровненного на границу выравнимания. То есть с 0x1000 или какого-то
 иного адреса. Размер выравнивания для секции - см. классе CIMAGE.

 Секции состоят из блоков данных класс CBLOCK. см. файл csection.cpp и .h

 Коментарий по PBD 
 -----------------

 Для того чтобы сохранить отладочную информацию для выходного образа, располагайте секции в том же месте виртуального пространства где они были в исходном.
